DATABASE IMPORT INSTRUCTIONS
----------------------------
1. Ensure XAMPP Apache and MySQL services are running
2. Open phpMyAdmin (http://localhost/phpmyadmin)
3. Click on the "Import" tab at the top
4. Click "Choose File" and select: database/alumni_db.sql
5. Click "Go" button at the bottom
6. The database 'alumni_db' will be created automatically - no manual creation needed!

TROUBLESHOOTING
---------------
If you still can't login:
1. Verify XAMPP MySQL is running
2. Import the SQL file via phpMyAdmin
3. Clear browser cache and cookies
4. Try accessing: http://localhost/alumni/admin/login.php
5. Use credentials: admin / admin123

For Coolify deployment issues:
1. Check environment variables are set
2. Visit /admin/test_connection.php
3. Verify MySQL service is running
4. Check database is imported
